# DalkkakAI

> Cloud-native multi-pane terminal multiplexer + Claude Code augmentation platform for solo indie hackers running multiple startups.

## Status

**v2 in development.** Old v1 (Python/FastAPI) preserved in [`old_repo/`](old_repo/).

## Quick start (v2)

```bash
pnpm install
pnpm dev
```

`apps/web` on :3000, `apps/api` on :8000.

## Run v1 (legacy)

```bash
cd old_repo
docker-compose up
```
